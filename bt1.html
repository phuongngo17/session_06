What is CSS?

Casscading Style Sheet (CSS) is used to fomat the layout of a webpage.

With CSS, you can control the colors, font, the size of text, the soacing between elements, how element are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much morel.
Tip:the word Casscadingmeans that a style applied to a parent element will also apply to all children elements within the parent. So, if you set the color of the body text to "blue", all headings, paragraphs, and orther text elements within the body will also get the same color (unless you specify something else)
